归并排序

  • 分-和, 时间复杂度O(nlogn), 空间复杂度O(n)

      def merge_sort(L):
      if len(L) < 2:
          return L

      mid = len(L) / 2
      left_L = merge_sort(L[:mid])
      right_L = merge_sort(L[mid:])
      return merge_array(left_L, right_L)


  def merge_array(left, right):
      print(left, right)
      new_L = []
      l = r = 0
      while len(left) > l and len(right) > r:
          if left[l] > right[r]:
              new_L.append(left[l])
              l += 1
          else:
              new_L.append(right[r])
              r += 1
      new_L.extend(left[l:])
      new_L.extend(right[r:])
      return new_L

插入排序

  • 时间复杂度 o(n**2), 比较麻烦的是数组的移动

      def insert_sort(L):
      for i, item_i in enumerate(L):
          index = i
          while index > 0 and L[index - 1] > item_i:
              L[index] = L[index - 1]
              index -= 1

          L[index] = item_i

      return L

堆排序

  • 堆排序: 大根堆,取走堆顶(实际是数组第一个元素),重新构建大顶堆,依次取走堆顶, 堆的存储结构是数组,为何还会有堆排序=>是一种选择排序,比直接的选择排序快

      def heapsort(x):
      if not x or len(x) < 2:
          return x
      a = []
      heapq.heapify(x)
      while len(x):
          top = heapq.heappop(x)
          a.append(top)
          heapq.heapify(x)

      return a

选择排序

  • 时间复杂度 o(n**2) 不稳定排序

      def select_sort(L):
      count = len(L)
      for i in xrange(count):
          min = i
          for j in xrange(i+1, count):
              if L[min] > L[j]:
                  min = j
          L[i], L[min] = L[min], L[i]

冒泡排序

  • 时间复杂度 o(n*n) 稳定排序. 持续比较相邻元素,大的挪到后面,因此大的会逐步往后挪,故称之为冒泡

      def bubble_sort(L):
      count = len(L)
      for i in xrange(count):
          for j in xrange(1, count - i):
              if L[j] < L[j - 1]:
                  L[j - 1], L[j] = L[j], L[j - 1]

二分查找

  • 最坏情况下，关键词比较次数为log2(n+1)，且期望时间复杂度为O(log2n)；

      # 递归
  def bisect(a_list, n):
      low = 0
      high = len(a_list)
      mid = (low+high)//2
      if a_list[mid] < n:
          bisect(a_list[mid+1:high], n)
      elif a_list[mid] > n:
          bisect(a_list[low:high], n)
      else:
          return mid


  # 非递归
  def bisect(a_list, n):
      low = 0
      high = len(a_list)
      while low < high:
          mid = (low+high)//2
          if a_list[mid] == n:
              return mid
          elif a_list[mid] < n:
              low = mid + 1
          else:
              high = mid
